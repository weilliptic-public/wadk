# Arithmetic MCP

This is a minimal Model Context Protocol (MCP) applet exposing simple
arithmetic utilities (`add`, `multiply`) for agentic workflows on WeilChain.

This MCP is presented as an example of thow MCP are defined and developped.
Its interface is defined in the `arithmetic.widl` and implemented in the `src/lib.rs` file.
Please refer to the `.widl` and `.rs` files for detailed documentation.
