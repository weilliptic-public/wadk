US Counties COVID-19 Data
- Location: `/databricks-datasets/COVID/covid-19-data/us-counties.csv`
- warehouse_id: c979ef6456a064c9
- Columns:
  - `date` (string): Date in YYYY-MM-DD format
  - `county` (string): County name
  - `state` (string): State name
  - `fips` (string): Federal Information Processing Standards code for county
  - `cases` (integer): Cumulative number of cases
  - `deaths` (integer): Cumulative number of deaths
  
- Example Query:
  ```sql
  SELECT state, county, MAX(cases) as total_cases, MAX(deaths) as total_deaths
  FROM read_files('/databricks-datasets/COVID/covid-19-data/us-counties.csv')
  WHERE state = 'California'
  GROUP BY state, county
  ORDER BY total_cases DESC
  LIMIT 10;
  ```

US States COVID-19 Data
- Location: `/databricks-datasets/COVID/covid-19-data/us-states.csv`
- warehouse_id: c979ef6456a064c9
- Columns:
  - `date` (string): Date in YYYY-MM-DD format
  - `state` (string): State name
  - `fips` (string): Federal Information Processing Standards code for state
  - `cases` (integer): Cumulative number of cases
  - `deaths` (integer): Cumulative number of deaths
- Example Query:
  ```sql
  SELECT state, MAX(cases) as total_cases, MAX(deaths) as total_deaths,
         ROUND(MAX(deaths) * 100.0 / MAX(cases), 2) as death_rate
  FROM read_files('/databricks-datasets/COVID/covid-19-data/us-states.csv')
  GROUP BY state
  ORDER BY total_cases DESC
  LIMIT 5;
  ```
