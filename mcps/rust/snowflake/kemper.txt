
Schema: TPCH_SF10 
Table: CUSTOMER
Description: Stores customer data.

| Column        | Description                                                            |
| ------------- | ---------------------------------------------------------------------- |
| c_custkey    | Customer Key (Primary Key)                                             |
| c_name       | Customer Name                                                          |
| c_address    | Customer Address                                                       |
| c_nationkey  | Foreign Key to the NATION table, indicating the customer's nationality |
| c_phone      | Customer Phone Number                                                  |
| c_acctbal    | Customer Account Balance                                               |
| c_mktsegment | Customer Market Segment                                                |
| c_comment    | Customer Comment                                                       |

Schema: TPCH_SF10
Table: LINEITEM
Description: Represents the individual line items within customer orders.

| Column           | Description                                                                    |
| ---------------- | ------------------------------------------------------------------------------ |
| l_orderkey      | Foreign key referencing o_orderkey in the ORDERS table                        |
| l_partkey       | Foreign key referencing p_partkey in the PART table                           |
| l_suppkey       | Foreign key referencing s_suppkey in the SUPPLIER table                       |
| l_linenumber    | Line number within an order (part of composite primary key)                    |
| l_quantity      | Quantity of the part ordered                                                   |
| l_extendedprice | Extended price (quantity Ã— unit price)                                         |
| l_discount      | Discount applied to this line item                                             |
| l_tax           | Tax applied to this line item                                                  |
| l_returnflag    | Single character flag indicating return status                                 |
| l_linestatus    | Single character flag indicating status (e.g., 'O' for open, 'F' for finished) |
| l_shipdate      | Date the line item was shipped                                                 |
| l_commitdate    | Date the line item was committed to be shipped                                 |
| l_receiptdate   | Date the line item was received by the customer                                |
| l_shipinstruct  | Shipping instructions                                                          |
| l_shipmode      | Shipping mode used                                                             |
| l_comment       | Descriptive comment                                                            |

Schema: TPCH_SF10
Table: REGION
Description: Contains geographical region information.

| Column       | Description                                                  |
| ------------ | ------------------------------------------------------------ |
| r_regionkey | Unique region key (Primary Key)                              |
| r_name      | Name of the region (max 25 chars, e.g., 'AMERICA', 'EUROPE') |
| r_comment   | Description or comment (max 152 chars)                       |


Schema: TPCH_SF10
Table: NATION
Description: Details about nations in the simulated business.

| Column       | Description                                              |
| ------------ | -------------------------------------------------------- |
| n_nationkey | Unique nation key (Primary Key)                          |
| n_name      | Name of the nation (max 25 chars)                        |
| n_regionkey | Foreign key referencing r_regionkey in the REGION table |
| n_comment   | Additional comments (max 152 chars)                      |


Schema: TPCH_SF10
Table: SUPPLIER
Description: Contains details about suppliers of parts.

| Column       | Description                                     |
| ------------ | ----------------------------------------------- |
| s_suppkey   | Unique supplier key (Primary Key)               |
| s_name      | Supplier name (fixed length 25 chars)           |
| s_address   | Physical address (up to 40 chars)               |
| s_nationkey | Foreign key to n_nationkey in the NATION table |
| s_phone     | Supplier phone number (15 chars)                |
| s_acctbal   | Supplier account balance                        |
| s_comment   | Additional comments (up to 101 chars)           |

Schema: TPCH_SF10
Table: ORDERS
Description: Contains customer order information.

| Column           | Description                                                                    |
| ---------------- | ------------------------------------------------------------------------------ |
| o_orderkey      | Unique order key (Primary Key)                                                 |
| o_custkey       | Foreign key referencing c_custkey in the CUSTOMER table                       |
| o_orderstatus   | Single character indicating status ('F' = finished, 'O' = open, 'P' = pending) |
| o_totalprice    | Total price of the order                                                       |
| o_orderdate     | Date order was placed                                                          |
| o_orderpriority | Priority of the order                                                          |
| o_clerk         | Clerk who processed the order                                                  |
| o_shippriority  | Shipping priority                                                              |
| o_comment       | Order-related comments                                                         |


Schema: TPCH_SF10
Table: PARTSUPP
Description: Availability and cost of parts from suppliers, linking PART and SUPPLIER.

| Column         | Description                                              |
| -------------- | -------------------------------------------------------- |
| ps_partkey    | Foreign key referencing p_partkey in the PART table     |
| ps_suppkey    | Foreign key referencing s_suppkey in the SUPPLIER table |
| ps_availqty   | Available quantity from supplier                         |
| ps_supplycost | Cost at which supplier provides the part                 |
| ps_comment    | Description or comments                                  |
